§ 02 · What It Does

Four modules, one command.

Run /morning and it fires all four in sequence. Skip any of them by editing the SKILL.md later — but the default runs the full brief.

1 · Email triage + drafts

Scans Gmail. Sorts into brand deals / important / noise. Auto-drafts replies to the partnership threads (with your manager CC'd). You wake up, review the drafts, hit send.

2 · News that matters

Pulls AI + Claude Code news by default — new releases, features, actionable stuff you can talk about on camera today. This is the module you'll rewire to your niche (real estate, fitness, ecom — page 5).

3 · Task priorities

Reads your active projects (git commits, calendar, open loops), returns your top 3 for the day. Not 12. Three.

4 · Account stats

Yesterday's numbers — follower delta, top-performing post, DMs. So you know what actually landed before you plan today.

Step 01

Install it in 30 seconds.

A Claude Code skill is one folder with one file. Do these three steps and you're live in every project you open.

  1. Open a terminal and run: mkdir -p ~/.claude/skills/morning
  2. Create a new file at ~/.claude/skills/morning/SKILL.md and paste in the starter skill on the next page.
  3. Open Claude Code in any project. Type /morning. If the slash menu autocompletes, it's installed.

The starter skill (part 1 of 2)

Copy this and the next page into your SKILL.md. It's the base version — you'll personalize it after install (page 5).

# Morning Briefing

Runs 4 modules in parallel and combines them into one brief.

## Config (edit these once)

- Your name: YOUR_NAME
- Timezone: YOUR_TIMEZONE
- News niche: AI + Claude Code   ← change to your niche
- Important email keywords: partnership, sponsor, collab, invoice, booking
- Manager CC (for partnership drafts): none

## Execution

Spawn all 4 agents in a SINGLE message so they run in parallel.
When they return, combine outputs into the final format below.

## Agent 1 — Email triage + drafts

Search Gmail for unread inbox mail from the last 48 hours (paginate all pages, don't stop at 50). Read every non-bulk email body. Categorize into:
- Important (real replies, calls booked, partnership threads)
- Revenue (payments, new customer notifications)
- Noise (marketing, alerts, digests)

For any thread matching the "Important email keywords" above, DRAFT a reply using gmail_create_draft. If a Manager CC is set, CC them on partnership drafts.

Return counts per bucket + a one-line note per Important thread.

## Agent 2 — News

Pull the last 24 hours of news in the configured News niche. Return 5 bullets max. Each bullet ends with a "so what for me today" line — one sentence, actionable.

## Agent 3 — Priorities

Read today's calendar events + the last 24 hours of git commits across the user's active repos. Return exactly the top 3 priorities for today. Not 12. Three.

## Agent 4 — Account stats

Return yesterday's numbers from tracked social accounts: follower delta, top-performing post, DM count. If no stats source is wired up, return "not configured — ask Karem to wire this."

## Final output format

# Morning Briefing — [DATE]

## Email
[Agent 1]

## News
[Agent 2]

## Top 3 Today
[Agent 3]

## Stats
[Agent 4]

---
End with: "What do you want to hit first?"
That's the whole install. No npm, no config, no restart. Fire /morning once to sanity-check it works — then move to page 5 to make it yours.

Step 02

One command unlocks it all.

This is the single most important page in this guide. If you skip it, you get my morning. If you use it, you get yours.

After you install the skill, open Claude Code and type this — exactly:

Tell me about my morning skill.

Claude will read the SKILL.md and explain the four modules — email triage, news, priorities, stats — the way I have them set up by default. Now you have leverage. You know what's in there, so you know what to change.

Why this matters

Every skill I ship is a starting point, not a fixed script. Once Claude has explained what it does, you can tell it to rewire any part of it in plain English — no code, no re-install, no messing with the file yourself. Claude edits the SKILL.md for you.

Rule of thumb: Any time you install a skill of mine, the first thing you should type is "Tell me about my [skill name] skill." Then tell Claude what to change. That's how you turn a shared skill into your skill.

Step 03

Swap the news for your niche.

The default news module pulls AI + Claude Code news because that's what I make content about. If you're a real estate agent, that's noise. If you sell skincare, that's noise. Rewire it.

The exact prompt

After you've run Tell me about my morning skill, follow up with something like this — swap the bracketed part for your actual niche:

I don't want AI news in the morning brief.
I want [real estate market news for the GCC region] instead.
Rewrite the news module in my morning skill so it pulls that
every morning, and keep the rest of the skill exactly the same.

Claude will edit the SKILL.md for you. Next time you fire /morning, the news module runs against your niche.

Some examples people have already rewired it for

Ecom operator

"Pull DTC + Meta ads news, Klaviyo product updates, and any Shopify feature launches from the last 24 hours."

Real estate agent

"Pull mortgage rate moves, new listing volume in my metro, and any local zoning or tax news."

Fitness coach

"Pull new research on training + nutrition, viral fitness content from the last 24 hours, and creatine/supplement news."

B2B founder

"Pull funding announcements in my space, competitor product launches, and any regulatory changes for SaaS in my market."

Same move works for every module. Don't want auto-drafted email replies? Say so. Want a different definition of "important"? Say so. The skill bends around your day, not the other way around.

Step 04

Fire it while you're asleep.

Running /morning manually is fine for week one. Then you'll get tired of typing it, and that's when you schedule it.

Run it once, manually

  1. Open Claude Code.
  2. Type /morning. Hit enter.
  3. Watch it triage, pull news, list priorities, drop stats. Read it. Fix anything wrong (via the prompt on page 5).
  4. Do this for 3 mornings until the output is exactly what you want.

Then schedule it

Claude Code supports scheduled tasks. Ask Claude:

Schedule my morning skill to run every weekday at 7am.

Claude sets up the schedule. From then on, the brief lands in your Claude Code session before you sit down. You open the laptop, the drafts are already written, the news is already summarized, the top 3 are already picked.

The whole point: your first hour stops being about processing input and starts being about executing on a plan that was already made for you.
